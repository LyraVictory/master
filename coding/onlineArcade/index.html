Online Arcade

The hub for playing online games. These include top-down and platformer games.

  • Mystery Mansion

    This game is made by Sunnydee1221.
    It is a kind of puzzle/platformer game.
    You are trying to get through all
    the different floors of a mansion.

  • Easy Street

    This is the first game I made.
    There are three different levels,
    each getting progressively harder,
    and each with different styles and objectives.

  • Out Of This World

    This is the second game I have made.
    It is a platformer that is more story-based.
    You are an astronaut going through
    different kinds of levels to complete.
    It has a very satasfying ending.

  • Sky Adventure

    This game is made by Giankr278.
    It is a short and sweet platformer game.
    It has only one level but it is fun and
    challenging. You are collecting coins
    and escaping all of the enemies.

  • Penguinball

    This game was made by my good
    friend Callie. It is a very simple game.
    You have to keep the ball from touching
    the ground by moving your character
    side to side and bouncing it on your head.
